LLM 기반 개인화 추천서 생성 및 품질 평가 시스템 설계

1. 문체 학습 기반 개인화

TXT, DOCX, PDF 업로드 시 텍스트를 추출하여 Claude API로 분석

어조, 말투, 문장 끝맺음 등 작성자 고유 문체 파악

분석 결과를 프롬프트에 반영해 개인화된 문체 재현

2. 추천 강도 및 톤 조절

1~5점 척도로 추천 강도 조절 가능

낮은 점수 → 사실 중심 나열

높은 점수 → 구체적·강력한 추천

톤 옵션 4가지 제공: 공식적, 친근한, 간결한, 설득형

강도 × 톤 조합으로 20가지 이상 차별화된 추천서 생성 가능
