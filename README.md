# LLM 기반 개인화 추천서 생성 및 품질 평가 시스템

> 작성자의 문체를 반영한 추천서를 생성하고, LLM 기반 품질 평가를 수행하는 AI 시스템

---
**[개인화 추천서 자동 생성 AI]**

지원자의 역량을 평가하는 추천서를 개인 맞춤으로 자동으로 생성하고 평가하는 AI 시스템입니다.


## 담당 역할

- 추천서 생성 파이프라인 설계
- Claude API 연동 및 프롬프트 설계
- GPT 기반 품질 평가 시스템 구현
- FastAPI 백엔드 개발
- React 프론트엔드 구현

**주요 기능**

## 주요 기능

### 1. 문체 기반 개인화 추천서 생성

- TXT, DOCX, PDF 업로드
- 기존 문체 분석
- 작성자 스타일을 반영한 추천서 생성

### 2. 추천 강도 및 톤 조절

- 추천 강도(1~5)
- 공식형
- 친근형
- 간결형
- 설득형

### 3. 품질 자동 평가

GPT 기반

- 정확성
- 논리성
- 개인화
- 전문성
- 설득력

### 4. 환각(Hallucination) 최소화

입력된 정보만 활용하도록 프롬프트를 설계하여 사실 기반 추천서를 생성

---
**시스템 구성**
-프론트엔드: React 18.3
-백엔드: Python FastAPI
-데이터베이스: MySQL/PostgreSQL
-생성 엔진: Claude Sonnet 4.5 (문장 안정성 및 긴 문맥 처리)
-평가 엔진: GPT-4 (객관적 품질 평가)

---

**성능**
본 시스템 평균 점수: 4.72/5
범용 LLM 대비 문체 재현과 환각 방지에서 우수
사용자 테스트에서 90% 긍정 평가
